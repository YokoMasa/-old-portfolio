Seriously Local Japan

日本には必ずしも有名ではない、現地の人しか知らないような美しい自然や神社仏閣などがたくさんあります。
そういった場所を日本に観光に来てくれる方々に紹介するためにこのサイトを作りました。

最近よくある、その場所に行ったこともないのにネットの拾い画像などで適当に書いている記事とは違い、
このサイトで紹介している場所にはすべて自分で訪れており写真もほとんど自分で撮ったものです。
(内容の薄い記事ほど良くSEOされていて信憑性のある記事を見つけるのが大変なのを最近感じており、微妙な気持ちになります。。)

主な使用技術: Spring boot, Vue.js, MYSQL

Vmstat Visualizer

オンプレミスのサーバを扱う時、AWSのメトリクスのグラフのように簡単にリソース状況を可視化できません。
このツールは、vmstatコマンドを実行したログからリソース状況をグラフ化し簡単に可視化することができます。

主な使用技術: HTML, chart.js

ラーメン屋ガチャ

ラーメンを食べたくなった時、我々はGoogleやGoogle Mapsで「ラーメン」と検索し価格、距離、評価なので一番良いラーメン屋を決めていると思います。
簡単に多くの情報にアクセスでき、合理的な判断が出来るのは情報社会の良いところだと思います。

しかし、時にはそんな多くの情報に触れ頭で処理することに疲れることもあります。
そんな時、このサイトは近所のラーメン屋をランダムで1軒選んで提示してくれますので、何も考えずにそのラーメン屋に行ってみてください。
ネットで他店と比較していたら行かなかったかもしれないラーメン屋が、実はものすごく美味しいラーメンを出しているかもしれません。

主な使用技術: Cloud Functions, Python, Netlify, JQuery
